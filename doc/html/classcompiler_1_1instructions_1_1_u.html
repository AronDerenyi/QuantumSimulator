Detailed Description

Applies a unitary gate to a qubit.

Gate transformation matrix: e^(-i(phi+lambda)/2)*cos(theta/2) -e^(-i(phi-lambda)/2)*sin(theta/2) e^(i(phi-lambda)/2)*sin(theta/2) e^(i(phi+lambda)/2)*cos(theta/2)
